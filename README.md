## Handmatige tests

### Keyboard
Tijdens het testen ben ik door middel van het tab knop op de keyboard door de hele pagina gegaan. Ik heb gezien dat er niks uitgelijnd was dus dat de pagina heel slecht was voor toegankelijkheid. 

#### Screen reader
Met een screen reader werkt de website goed. Alleen sommige dingen worden op een verkeerde volgorde uitgesproken. Dus dat moet wel aangepast worden, door de rest werkt t prima.

### Interactive elements
Bij het hoveren op knoppen werken er sommige knoppen wel maar alleen op hover en bijvoorbeeld niet op active of andere elementen die je kan plaatsen op een knop. Sommige knoppen werken helemaal niet en die krijgen ook geen hover animation.


### Headings and landmarks
Op de websites kreeg ik een goeie uislag voor de website, maar niet voor alles. Er was slecht gebruik gemaakt met h1, h2, h3 etc etc.

Dit waren mijn test resultaten voor ZVC-veenendaal.
